# Documentation

`shap` is a shell-native interface for ACP coding agents. Start with the README for the overview, or
jump straight to a guide below.

## Guides

- [Getting started](./getting-started.md) — from a fresh install to your first agent reply, plus a
  tour of every command.
- [Installation](./installation.md) — install from source with cargo or via the Nix flake, and
  verify it works.
- [Agents](./agents.md) — configure one or more ACP agents, models, and adapter-specific passthrough.
- [Configuration](./config.md) — the `config.toml` reference: fields, defaults, file locations, and
  state.
- [Shell integration](./shell-integration.md) — optional Zsh `:` commands, prompt segment, and the
  `:commit` widget.
- [Nix flake](./nix.md) — reproducible dev shell, package build, and `nix run`/`nix profile`.
